# MCMUGANE Trading System v1.0

알파카(Alpaca) API를 활용한 자동/수동 주식 매매 시스템

## 기능

- 📊 실시간 포트폴리오 관리
- 🤖 자동매매 전략 실행
- 💻 터미널 기반 수동 매매
- 📈 다양한 매매 전략 지원
- 🔒 Paper/Live 모드 지원

## 설치 방법

### 1. 환경 설정
`.env` 파일을 생성하고 Alpaca API 키를 입력:
```bash
ALPACA_API_KEY_ID=your_key_here
ALPACA_API_SECRET_KEY=your_secret_here
